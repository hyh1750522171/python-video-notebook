## 简介

大部分内容来自网络。

默认安装了 `Python 2.7` 本次使用 `python3.7.2`，以及相关的第三方包 `ipython`， `numpy`， `scipy`，`pandas`。

> life is short. use python.

笔记内容使用 `jupyter notebook` 来展示。

安装好 `Python` 和相应的包之后，安装jupyter：

```shell
$ pip3 install jupyter
```

命令行下输入：

```
$ jupyter notebook
```

来进入 `jupyter notebook`。
